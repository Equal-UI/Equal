#  Install

```bash
# npm
npm install equal-vue
```

```bash
# yarn
yarn add equal-vue
```

#  Usage

## All components

```js
import Vue from 'vue'
import Equal from 'equal-vue'
import 'equal-vue/dist/equal.css'

Vue.use(Equal)
```

## Or individual components

```js
import Vue from 'vue'
import { Button, Alert, Switch } from 'equal-vue'
import 'equal-vue/dist/equal.css'

Vue.use(Button)
Vue.use(Alert)
Vue.use(Switch)
```
